# Ben Kvale - Personal Command Center

An Xbox-inspired, gamified personal portfolio website featuring an interactive "quest" system and blade-style navigation. 

**Live Site:** [https://benkvale.com](https://benkvale.com)

## Features

- **Retro-Future UI:** Custom "Command Center" aesthetic with neon accents and glowing effects.

- **Quest System:** Career history and skills framed as "Active" and "Completed" quests with XP metrics.

- **Responsive Character Model:** Dynamic hero section with a 3D character model that adapts to screen size.

- **Formspree Integration:** Functional contact form ("Send Transmission") without backend code.

- **Fully Responsive:** Optimized layouts for desktop, tablet, and mobile devices.

## Tech Stack

- **Core:** Pure HTML5, CSS3, and Vanilla JavaScript.

- **Hosting:** Vercel (Auto-deployed via GitHub).

- **Styling:** CSS Grid/Flexbox, CSS Variables, and Keyframe animations.

## Deployment Workflow

This project is linked to Vercel. 

1. **Development:** Make changes locally to `index.html` or `style` blocks.

2. **Deploy:** Push changes to the `main` branch on GitHub.

3. **Live:** Vercel automatically detects the commit, builds, and deploys to `benkvale.com`.

## Local Development

To preview changes before pushing:

```bash
# Using Python (Run in project root)
python -m http.server 8000
```

Then visit http://localhost:8000 in your browser.
